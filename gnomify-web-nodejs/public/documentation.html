Getting Started

To use the Gnomify API, you must be a registered Gnomify user. If you haven't yet, sign up here.

Requests to the API must be authenticated using Basic Auth with your Gnomify username and password.

Upload file

POST https://api.Gnomify.com/upload 
curl -F file=@myvideo.mp4 -u username:password https://api.Gnomify.com/upload
Don't resize
curl -F file=@myvideo.mp4 -u username:password https://api.Gnomify.com/upload?noresize
Remove sound
curl -F file=@myvideo.mp4 -u username:password https://api.Gnomify.com/upload?mute

Import file from URL

GET https://api.Gnomify.com/import 
curl -u username:password https://api.Gnomify.com/import?url=https://archive.org/download/Windows7WildlifeSampleVideo/Wildlife.wmv
Don't resize
curl -u username:password https://api.Gnomify.com/import?url=https://archive.org/download/Windows7WildlifeSampleVideo/Wildlife.wmv&noresize
Remove sound
curl -u username:password https://api.Gnomify.com/import?url=https://archive.org/download/Windows7WildlifeSampleVideo/Wildlife.wmv&mute
